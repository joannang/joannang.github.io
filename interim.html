Interim: A Productivity Web Application

Interim was made as part of project under the Summer Orbital Programme run by the NUS School of Computing targeting Year 1 students. The application is based off the Pomodoro Method, where it is theorised that studying or working in 25 minute blocks would make individuals more productive. This was the first Web Application I had ever created, made using HTML, CSS, JavaScript and Google Firebase, all of which was self-learnt from scratch.
