Easy Zippy: A Capstone Industry Project

This project was undertaken under the Information Systems Capstone Project, an intensive 3 months grind to build a system from the ground up, together with accompanying documentation which included a full business analysis of the project to back up any technical decisions made.

My team partnered with a locker services company, Easy Zippy, to help achieve their objectives of pivoting the business. The resulting system had 4 Front-end Applications: 2 Web Applications, 1 Mobile Application, and 1 Kiosk Application. It was built using Node.js, React, and React Native. For this project, I focused primarily on the Web Applications, which targeted both the Staff and Merchant stakeholders.
