shopPAWholic: A one-stop pet goods & services marketplace

shopPAWholic was a project done under the Enterprise Systems Interface Design and Development Module. The system consists of two Front-end Applications: a Web Application and a Mobile Application. The Web Application was made using Java and JSF technology, while the Mobile Application was created using Ionic. Both applications were connected to a Java EE back-end. I worked mainly on the back-end and also on the Mobile Application, where both customers and sellers could sign up and buy or list/sell products respectively, among many other features.
